# Number

A number prettifier, outputs a slanted seven-segment display like
ascii-art version of your number.

## Example

```sh
$ date +"%H %M %S" | ./number
              /
              \/\
            /\  /
            \ \
             \/

        /\
         /\
       \  /
        \


   \
    \
/\
 /
 \/
```
